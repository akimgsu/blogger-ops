8. CSS Variables and Dark Mode

As CSS has evolved, you no longer need preprocessors like SASS or LESS just to use variables. We now have native CSS Custom Properties (often simply called CSS Variables). This technology completely transforms how we handle themes, making the implementation of features like Dark Mode incredibly efficient while minimizing the need for heavy JavaScript intervention.

Declaring and Using Native CSS Variables

CSS variables always start with two dashes (--). To make them globally available, you typically declare them inside the :root pseudo-class. To use them, you call the var() function.

/* Global Variable Declaration */
:root {
  --main-bg-color: #ffffff;
  --main-text-color: #333333;
  --primary-accent: #007bff;
}

/* Applying the Variables */
body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}

button {
  background-color: var(--primary-accent);
}

By using variables, if your company decides to undergo a massive rebranding and change its primary color, you don't need to hunt through dozens of CSS files. You just change one line in the :root block.

Implementing Dark Mode with Minimal to Zero JS

Where CSS variables truly shine is in theme switching. If you want the browser to automatically detect and apply the user's Operating System dark mode preference, you can use the @media (prefers-color-scheme: dark) query.

/* Override variables when the user's system prefers dark mode */
@media (prefers-color-scheme: dark) {
  :root {
    --main-bg-color: #121212;
    --main-text-color: #e0e0e0;
    --primary-accent: #bb86fc;
  }
}

If you want to give users a manual "Theme Toggle Button" on your site, you just need a tiny snippet of JavaScript to toggle a single data-theme="dark" attribute on the <html> or <body> tag.

/* Controlling dark mode via a data attribute */
[data-theme="dark"] {
  --main-bg-color: #121212;
  --main-text-color: #e0e0e0;
}

With this architecture, you don't need JavaScript to manually update the color of every single element on the page. JavaScript simply flips a switch at the top level, and CSS smoothly handles the entire color palette swap underneath via variable reassignment.
